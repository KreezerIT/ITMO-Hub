# *Модель Кронига–Пени для одномерного кристалла*

На основании модели Кронига–Пени промоделировать зонную структуру одномерного кристалла.

## Задачи

1. Проанализировать изменение ширины запрещённых зон для двух крайних случаев:
    - когда электрон является совершенно свободным;
    - когда электрон заперт внутри одной потенциальной ямы (стенки непроницаемы).

2. Рассмотреть также промежуточные случаи между этими крайними режимами.

## Потенциальная энергия

Периодический потенциал имеет вид:

$$
V(x) =
\begin{cases}
U, & nc + a < x < (n + 1)c \\
0, & nc < x < nc + a
\end{cases}
$$

где:
- `a` — ширина потенциальной ямы;
- `c` — постоянная кристаллической решётки;
- `n` — целое число (нумерация периодов):  
  $$ n=0,\pm1,\pm2,\ldots $$
