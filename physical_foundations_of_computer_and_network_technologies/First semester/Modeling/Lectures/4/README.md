# Задание к лекции №4. *Визуализация абсолютно упругого нецентрального соударения двух тел с разными массами*

Тела находятся в ограниченной прямоугольной
оболочке, соударения с оболочкой также считать абсолютно упругими.

Входные параметры: массы тел, начальные скорости (модуль и направление),
размеры оболочки.

Итоговый вид модели: динамическое столкновение тел друг с другом и с
оболочкой.
