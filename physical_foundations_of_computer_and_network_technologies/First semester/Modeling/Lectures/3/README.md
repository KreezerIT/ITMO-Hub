# Задание к лекции №3. *Визуализация движения точки на ободе колеса/диска*

Входные параметры:
радиус колеса, скорость центра масс. 

Итоговый вид модели: траектория
точки на ободе колеса (динамическая модель (преимущественно) или
статическая). Под динамической моделью понимается отрисовка кривой с
течением времени, статическая – график зависимости y(x) – траектория точки.
