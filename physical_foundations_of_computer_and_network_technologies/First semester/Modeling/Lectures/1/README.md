# Задание к лекции №1. *Перевод систем координат*

Написать программу, в виде исполняемого файла, которая переводит координат из одно
системы координат в другую.
- В описании программы должны быть указаны формулы перехода от одной системе к
другой.
- Входные параметры: координаты в одной СК, точность вычислений (количество знаков
после запятой).
