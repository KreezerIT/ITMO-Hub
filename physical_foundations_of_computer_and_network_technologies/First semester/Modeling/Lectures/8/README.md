# Задание к лекции №8. *Энергетические превращения при колебании груза на пружине*

Входные данные: масса груза, коэффициент жесткости, коэффициент сопротивления среды (сила сопротивления пропорциональная скорости). 

Итоговый вид: графики зависимости кинетической,
потенциальной и полной механической энергии от времени.
