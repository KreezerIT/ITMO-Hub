# "Физические основы компьютерных и сетевых технологий"

**Лабораторная работа** - эксперимент, по техническому описанию которого нужно самостоятельно в лабораториях университета 
настроить необходимые приборы и произвести замеры/вычисления. После чего составить отчет о проделанной работе
и защитить преподавателю

**Моделирование** - задание, в рамках которого нужно оцифровать какой-то физический эксперимент или расчеты (кодом) и защитить преподавателю

**Экзамен**  
У ***некоторых*** лекторов есть опция зарабатывания баллов за экзамен в течение семестра:
- [<u>соло</u>] Моделированиями/Задачами после почти каждой лекции, некоторые из которых вмещали в себе отработку материала нескольких лекций *(2-й семестр физики у автора)*
- [<u>команда</u>] Объемным заданием, по которому нужно собственноручно собрать физическую установку (*~~вероятно за свой счет~~*)
- [<u>команда</u>] Объемным проектом, в котором нужно сделать сайт/приложение для демонстрации физических свойств/явлений, который можно использовать в будущем. Например:
  - Русский аналог [IBM-калькулятора](https://quantum.cloud.ibm.com/composer)
  - [Сайт с наглядной демонстрацией телепортации кубита](Third%20semester/FinalExamProjectPictures) *(3-й семестр физики у автора)*
