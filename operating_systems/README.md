# "Операционные системы"

Вся практическая часть курса заключается в сдаче лабораторных работ и написании письменных тестов

ОС, использовавшаяся для выполнения работ - Linux Ubuntu 24.10

**Лабораторная работа** — техническое описание скриптов, которые нужно написать; 
либо описание эксперимента, который нужно проделать в среде Linux, после чего защитить преподавателю

**Тест/Опрос** — работа в формате письменного задания с вопросами по материалам прошедших лекций.  
Вопросы могут быть как теоретическими (например, дать определение, проанализировать скрипт), так и практическими, например — построить диаграмму выполнения процессов.  
Определения терминов ждут чаще всего такие, какие проговаривались на лекциях

Сдача лабораторных работ проходит **на своем ноутбуке**, могут попросить заново запустить написанные скрипты :)

Условие шестой лабораторной работы (хард) есть в двух вариантах, по желанию

***Некоторые*** преподаватели разрешали взять индивидуальное задание в счет *5/6* ИЛИ *6* работы,
по типу написать (код) какой-то прикладной проект (необязательно тесно связанную с ОСями)
