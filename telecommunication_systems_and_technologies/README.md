# "Телекоммуникационные системы и технологии"

Вся практическая часть курса заключается в сдаче лабораторных работ и прохождении онлайн-тестов

ОС, использовавшиеся для выполнения работ - Linux CentOS 9 и Linux Debian 12

**Лабораторная работа** - ТЗ, по которому нужно пошагово выполнить какой-то сценарий/эксперимент; написать скрипты; поработать с сетевыми утилитами/приложениями
- защита каждой работы преподавателю
- защита 7 работ + в сумме >=80 баллов ~ `5`
- защита 6 работ + в сумму >=68 баллов ~ `4`
- защита хотя бы 4 работ + в сумме >=44 балла + сданы обязательные 1,4,5 работы ~ `3`
- есть дедлайны с понижением баллов по 2 за неделю, максимум потеря не больше 6 баллов за работу

**Тесты** - список вопросов с одним/несколькими вариантами ответа (онлайн-форма на **fitp.itmo.ru**), являются контрольной точкой
- ≈50 равноценных вопросов
- имеет минимальный порог 60% верных ответов для зачтения результата
- при нехватке баллов до порога - можно переписать   
- у некоторых преподавателей возможно очно

*Экзамен для добора баллов, можно и без него закрыться на 5 автоматом*
